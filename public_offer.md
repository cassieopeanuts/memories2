## 1. ТЕРМИНЫ И ОПРЕДЕЛЕНИЯ

Для целей настоящей Оферты нижеприведенные термины используются в следующем значении:
* **Сервис (Сайт)** — интернет-ресурс, размещенный в сети Интернет по адресу `https://легкосохранить.рф` (включая все поддомены и связанные приложения), представляющий собой программно-аппаратный комплекс для хранения, сортировки и обмена цифровыми фотографиями и изображениями.
* **Администрация** — правообладатель Сервиса, осуществляющий его техническую поддержку, администрирование и обеспечивающий предоставление услуг.
* **Пользователь** — полностью дееспособное физическое лицо либо уполномоченный представитель юридического лица, прошедший процедуру авторизации на Сайте и принявший условия настоящей Оферты.
* **Облачное хранилище (Сейф)** — дисковое пространство на серверах инфраструктурного партнера Администрации (Selectel S3 и/или иных аналогичных провайдеров), выделенное Пользователю для размещения цифровых файлов.
* **Личный кабинет** — защищенный персональный раздел Пользователя на Сайте, доступ к которому осуществляется после прохождения авторизации и ввода PIN-кода.
* **PIN-код** — четырехзначный секретный цифровой код, устанавливаемый Пользователем при регистрации, используемый для авторизации на конкретном устройстве и защиты персональных данных Пользователя от несанкционированного доступа.
* **Тарифный план (Тариф)** — размер платы за предоставление дискового пространства определенного объема на определенный период времени (Подписка).

## 3. РЕГИСТРАЦИЯ, АВТОРИЗАЦИЯ И БЕЗОПАСНОСТЬ PIN-КОДА

3.1. Для использования Сервиса Пользователь должен пройти процедуру авторизации с помощью одного из предложенных способов:
* Авторизация через внешние сервисы идентификации (SSO): Яндекс ID, Сбер ID, Т-Банк ID.
* Авторизация путем ввода адреса электронной почты (Email) и ввода проверочного кода / перехода по одноразовой ссылке.
3.2. При первой регистрации Пользователь обязан задать уникальный четырехзначный PIN-код.
3.3. **Важные условия безопасности:**
* Пользователь несет единоличную ответственность за конфиденциальность своего PIN-кода.
* Администрация не имеет доступа к PIN-коду Пользователя, не хранит его в открытом виде и не осуществляет его восстановление в случае утери.
* Вся активность в Личном кабинете, осуществленная после успешного ввода PIN-кода, признается действиями самого Пользователя.
3.4. Сервис использует механизм сохранения сессии (persistent session lock). При закрытии вкладки браузера сессия остается активной на устройстве Пользователя, но при повторном входе блокируется экраном ввода PIN-кода. Для полной очистки сессии Пользователь должен нажать кнопку «Выйти».

* Денежные средства за активированный и частично неиспользованный период Подписки возврату не подлежат.
